PROJECTS

Home Automation System

The project was to perform a design of an ergonomic system implementing using the knowledge of Electrical and Computer Engineering implemented by the language of Arduino. I designed a home automation system using a specific connections of sensors and actuators that performs some processes to make a more comfortable life to the users.

Cache Memory Simulator

The Cache Memory Simulator recreates a simulation of cache using the MIPS assembly language utilizing the specifications established by the professor. The purpose of the project was to allocate some random memory directions into the code using a memory size of 32, mapping function Fully Associative and FIFO replacement algorithm.

Faculty Evaluation System

The Faculty Evaluation System involved to elaborate a system with integrated GUI interface using Java language. The structure of the Software consisted of a connection between some GUI interfaces with integrated database and a logical code.
